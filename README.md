# Exploratory Data Analysis (EDA) Project

## Objective
Analyze student performance data and identify patterns and trends.

## Dataset
Students Performance Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Descriptive Statistics
- Missing Value Analysis
- Correlation Analysis
- Distribution Analysis
- Outlier Detection

## Visualizations
- Histogram
- Correlation Heatmap
- Box Plot

## Key Findings
- Strong correlation between reading and writing scores
- Most students scored between 50 and 80
- Minimal outliers observed
